My Experience Learning Web Development

The course I chose to study at University is Business Computing, one of the modules for the subject is Web Development. I have now studied this module for a whole term and I am very much enjoying it as I take a keen interest in this hence why I chose it as my selected pathway for the course.

Starting of learning web design was difficult for me, I struggled to remember the tags needed and always forgot to close the tags. To overcome this, I went to more lectures and done my own independent research using YouTube and other websites. This helped to increase my understanding for coding, which led me to be more confident in how HTML/CSS coding works.

I can type the majority of the page HTML code fluently. Examples of codes I didn’t remember before but remember now are the hyperlink tag, which allows me to link text to another page (a href=" ">.) Next is the heading tag, this creates a heading and the size of the heading can be changed by adjusting the number in the tag. The higher the number the smaller the text will get (h1-5.)

In conclusion, I am enjoying learning this module and wish to continue developing my knowledge on how to code HTML/CSS as this will be very beneficial for my future employment.

My Design

Above are three pictures that show my draft design, my intial design and my final design. The first picture on the far left is my draft design. This is how I originally intended my website to look like. I decided that I wanted the basic header and footer design but to have the navigation menu on the side of the document. I figured this would be a convenient and open place to put it so the users will find it easy to navigate around the site.

The middle one is the design that I changed my draft to. I changed it to this one because it had an increased header and footer height, which allows me to include more information in the footer. I also wanted the header to stand out more.

Finally, the one on the far right, this my third and final design. This is the concept that I kept and am very pleased with. I kept the increased header and footer design as I still believe this is beneficial. However, I decided to move the navigation bar to the header. I believe this this is simpler and looks neater, it also creates more space on the body of the web page.

Site Validation

After finishing designing my website I checked my code for each page ensure it was all correct and their was no errors. I done this for my HTML and my CSS code. My HTML code for each page came back the same five errors. whereas my CSS check came back with zero errors, which I am very pleased with. Validation your HTML/CSS code is very important as you can correct any errors that are on the page, it's also good as you'll be less likely to make the same mistake again in the future.
