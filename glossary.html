Glossary

When learning Web Development there are many new terms that we have come accross. Below is the majority of terms that have been learnt whilst learning this module.

A

Alignment - This is the positioning of elements in a coding design. For example, text and images. They can be aligned to each other or to a certain place on the page.

Animation - These are images that simulate movement when placed in a specific order.

Attribute - This is added to a HTML start tag in HTML syntax.

C

Class - This attribute allows you to assign to it many subtypes.These subtypes may then be used in CSS code for styling purposes, or by JavaScript code.

Commenting - You can add comments to HTML documents, it will not be desplayed but can help you remember why you done something a particular way.

CSS - Stands for Cascading Style Sheets. CSS is a language used to style and design the webpage and the contents on it. It can change the: size, font, font colour, spacing and background colour/images.

D

Declaration - A declaration determines the name and data type of a variable/element. You can declare variables by writing the name of the variable into code, as well as data type indicators and other required syntax.

F

File Path - The file path is a unique location in a file system. It creates a pathway by following the directory tree hierarchy.

Font - This is text written in a specific style and size. An example of this is Calibri(Body).

Footer - A footer holds all extra information that you may want the customer to know, at the bottom of the page.

G

GIF - Stands for Graphics Interchange Format. This is a format used to store compresed image files.

H

Header - This runs along the top of a webpage. It's main purpose is to promote the company's brand and make it more recognisable.

Home Page - This is the main page of a website and the first page that you will see when you enter a url link into the web browser. The page that is displayed by default is index.html

Href - The HREF is an attribute that contains two components: the URL, which is the actual link, and the clickable text that appears on the page.

HTML - Stands for Hyper Text Markup Language. HTML is the main language used in the world wide web and is also used as a coding language to construct these web pages. For example, this one.

HTML Document - This is a document that is written in HTML. HTML document is synonomous with HTML page.

HTML Editor - This is a software tool used to assemble webpages.

HTML Form - HTML form is a webpoage that is filled out by the user, which then sends data to the web server.

HTML Page - A document that is written using HTML. HTML page is synonomous with HTML document.

HTML Tags - Pieces of code that are used to identify what is to be displayed on a HTML document. For example, 'p' this a paragraph tag and is used so the HTML editor knows to structure it in a poaragraph format.

HTTP - Stands for Hyper Text Transfer Protocol. It sends data from the web server to the web browser.

HTTPS - Stands for Hyper Text Transfer Protocol Secure. HTTPS is a secure version of HTTP as the data that is sent is encrypted.

Hypertext - This is text that contains links to other text.

Hyperlink - A link on a webpage, that links to another web page. The link can be in the form of an image or text

I

Icons - These are placed on web pages and often represent the topic or information category to another Web page.

Img - This is the element used in HTML syntax to embed a image to a webpage.

J

Java - A programming language used to write applications that can be run on web pages. Can also be used to make other things like games etc.

JavaScript - A programming language used for web pages that can be used to allow users to interact as well as create moving images and clocks.

JPEG - Stands for Joint Photographics Experts Group. This is a compresion format that is used for images.

L

Landing page - A landing page is the page that users will first see when they enter a website, like the index/home.

Link - This is a relationship between two things. Clicking a link will take you to a different page.

N

Navigation - Navigation refers to the system that allows users move around and get to different pages of the website easier. This is often done through menus.

Nested elements - This means that the tag opened most recently is going to be the next tag to close.

M

Meta Tags - A meta tag is a special tag used to describe some of the contents on a web page.

P

PDF - Stands for Portable Document Format. This is a format developed by Adobe Systems and allows its users to view formatted documents in a way that was intended by the creator.

PNG - Stands for Portable Network Graphics. PNG is a graphics format that compresses images. It is said to replace GIFs in the future.

Properties - Ways in which you can style a given HTML element in the CSS page.

Pseudo Class - This is used to define the special state of a selected element.

R

Root Directory - The root directorry is the beggining of a folder structure and the highest directory in the heirarchy. It contains all other folders in the drive as well as files. For example, the root directory on this PC would be C:\.

S

Sandbox - A sandbox is an isolated environment that can test out code to try it out before putting it into production.

Script - This is the part of code that makes the HTML page more dynamic & interactive. One type of script is Javascript.

Serif - A small line attached to the end of a printed word in some fonts. 'Sans serif' refers to fonts that don't have this line.

Src - The SRC attribute is used to include some external entities like images, CSS file, HTML file, any other web page or a JavaScript file.

Syntax - Syntax is the grammar, structure, or order of the elements in the computer language.

T

TAG - A tag is a set of markup characters that are used around an element to indicate its start and end.

Text Editors - An HTML editor is a computer program for editing HTML, it can be used to create web pages.

Transform - The transform CSS property lets you modify elements. it allows elements to be translated, rotated, scaled, and skewed.

Transitions - This is the animation that takes place when changing CSS properties. Instead of having property changes take effect immediately, you can cause the changes in a property to take place over a period of time and with effects.

Typography - This is the style and appearance of printed type. It is an art and can be edited to make the text more appealing when displayed.

U

UI - This stands for user interface. This is everything designed into an information device where the user may interact.

V

Validator - In simple terms, validation ensures that your website complies with the standards accepted by most web designers.
